Introduction

ggplot2 = un package hyper fashion.

ggplot2 est utilis pour raliser des graphiques selon une philosophie qui lui est propre (en loccurrence, inspire du livre The Grammar of Graphics de Leland Wilkinson, do son nom ggplot), et qui le distingue des autres outils de production graphique sous R, notamment les fonctions graphiques de base comme hist, boxplot, etc.

Lide gnrale est de dcrire et donc produire un graphique comme un assemblage de couches

=> production simple et intuitive de graphiques beaux et riches

Commenons par charger le package ggplot2 -aprs lavoir, si ncessaire, install- :

require(ggplot2)

Les bases: jeu de donnes, aethetics, et geoms

On cre un graphique laide de la fonction ggplot. On spcifie sur quel jeu de donnes le graphique va tre construit, ainsi que les aesthetics x et y:

p <- ggplot(catdata,aes(x=haircolor,y=weight))
plot(p)

Choisissons par exemple de reprsenter linformation laide dun geom point:

p <- ggplot(catdata,aes(x=haircolor,y=weight)) +
  geom_point()
plot(p)

Modification du type de geom

Si je souhaite raliser un graphique de type boxplot plutt que point alors il faut que je modifie le type de geom que jutilise :

p <- ggplot(catdata,aes(x=haircolor,y=weight)) +
  geom_boxplot()
plot(p)

Remarque

Vous devez choisir un type de geom compatible avec la nature des informations (en x et y) que vous voulez reprsenter!

p <- ggplot(catdata,aes(x=haircolor)) +
  geom_point()
plot(p)
## Error: geom_point requires the following missing aesthetics: y

Superposition de geoms

p <- ggplot(catdata, aes(x=haircolor, y=age))+
  geom_point(col="blue", size=2) +
  geom_jitter(col="pink", size=2)
plot(p)

Paramtrisation du geom

Evidemment tous les geoms peuvent tre paramtrs. Je peux par exemple modifier la couleur de mes botes moustache de la faon suivante:

p <- ggplot(catdata,aes(x=haircolor,y=weight)) +
  geom_boxplot(fill="lightskyblue")
plot(p)

Parmi les arguments les plus utiliss pour paramtrer les geoms, on peut citer:

  • color: la couleur des lignes
  • fill: la couleur de remplissage
  • linetype: le type de ligne
  • size: la taille des symboles ou lpaisseur des lignes
  • alpha: la transparence

Paramtrisation du geom

p <- ggplot(catdata, aes(x=age, y= weight)) +
  geom_point(shape=24,
             color="blue",
             fill="lightskyblue",
             alpha=0.6,
             size=3)
plot(p)

p <- ggplot(catdata, aes(x=age)) +
  geom_density(color="red",
               fill="pink",
               linetype=3,
               lwd=2)
plot(p)
